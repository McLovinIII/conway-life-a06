## Назначение

Настольное приложение моделирует клеточный автомат Conway's Game of Life по правилу B3/S23. Пользователь может рисовать начальную конфигурацию, запускать поколение вручную или автоматически, менять скорость, загружать и сохранять состояние поля.

## Запуск

```bash
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
python main.py
```

## Тесты

```bash
pytest
```
